# 09 - Banco Mestre de Máquinas

## Objetivo

Definir o cadastro técnico padronizado de máquinas e equipamentos.

## Estrutura de cada registro

-   ID
-   Nome
-   Categoria
-   Fabricante
-   Modelo
-   Atividades relacionadas
-   Perigos associados
-   Controles existentes
-   EPCs
-   EPIs
-   Inspeções
-   Manutenção
-   Treinamentos
-   Documentos gerados
-   Referências
