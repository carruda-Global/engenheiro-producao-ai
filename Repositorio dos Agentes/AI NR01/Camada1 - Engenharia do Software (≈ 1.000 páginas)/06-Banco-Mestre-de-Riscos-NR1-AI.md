# 06 - Banco Mestre de Riscos NR1 AI

## Objetivo

Definir a estrutura padronizada de todos os perigos e riscos utilizados
pelo sistema.

## Estrutura do Registro

-   ID
-   Código interno
-   Categoria
-   Perigo
-   Agente
-   Descrição
-   Fonte geradora
-   Atividades relacionadas
-   Funções relacionadas
-   Setores relacionados
-   Máquinas relacionadas
-   Produtos químicos relacionados
-   Consequências
-   Probabilidade
-   Severidade
-   Nível do risco
-   Medidas de eliminação
-   Medidas de substituição
-   Controles de engenharia
-   Medidas administrativas
-   EPIs
-   EPCs
-   Treinamentos
-   Documentos gerados
-   Referências técnicas
-   Versão

## Categorias

-   Físicos
-   Químicos
-   Biológicos
-   Ergonômicos
-   Acidentes

## Exemplo

### R-0001

Perigo: Ruído contínuo

Categoria: Físico

Atividades: - Operação de compressores - Serralheria - Marcenaria

Consequências: - PAIR - Estresse - Fadiga

Controles: - Enclausuramento - Manutenção - Redução da exposição

EPI: - Protetor auricular

Documentos: - Inventário - Plano de Ação - PGR

## Regras

Todo risco deverá possuir: - pelo menos uma medida de controle; - pelo
menos um documento associado; - vínculo com uma categoria; - vínculo com
atividades aplicáveis.
